what is flex ?

Flexbox is a CSS module that offers a flexible container layout system. It enables you to create a flexible, responsive layout by using a single, flexible container element and its flexible items.

CONTAINER Properties -

display: flex

The display property is used to specify the type of box generated by an element. By default, all elements are displayed as inline boxes. By setting display: flex, the element becomes a flex container.

flex direction

The flex-direction property determines the direction in which flex items are laid out. It can have the following values:

  • row - The flex items are laid out in a row from left to right.

display flex properties

Flex items can be aligned along the main axis and cross axis using the flex-direction property.

  • justify-content - Defines how flex items are positioned along the main axis.
  • align-items - Defines how flex items are positioned along the cross axis.


justify content and align items are not dependent on vertical or horizontelness it depends on axis - flex items are always laid out in the main axis.

cross axis and main axis

The main axis and cross axis are determined by the flex-direction property. The main axis is the horizontal axis, and the cross axis is the vertical axis.

  • main axis - The direction in which flex items are laid out along the main axis.
  • cross axis - The direction in which flex items are laid out along the cross axis.

main and cross start and end

The main start, main end, cross start, and cross end properties define the position of flex items along the main and cross axes respectively.

flex wrap : wrap property

The flex-wrap property determines whether flex items should wrap onto multiple lines or not. It can have the following values:

  • nowrap - The flex items do not wrap onto multiple lines.
  • wrap - The flex items wrap onto multiple lines.
  • wrap-reverse - The flex items wrap onto multiple lines, but in reverse order.

    alignself

    The align-self property overrides the align-items property for individual flex items. It can have the following values:

  • auto - The default value. The flex item aligns itself with the parent's align-items property.
  • flex-start - The flex item aligns itself at the start of the cross axis.
  • flex-end - The flex item aligns itself at the end of the cross axis.
  • center - The flex item aligns itself in the center of the cross axis.
  • stretch - The flex item stretches to fill the container.
