Animation in css

CSS animations allow you to define complex animations using CSS rules. They can be used to create effects like fading, sliding, scaling, rotating, and more.
To create an animation, you define a keyframe animation using the @keyframes rule. Then, you apply the animation to the desired element using the animation property.
Here's an example of a simple animation that changes the background color of a paragraph element:
@keyframes colorChange { 0% { background-color: red; } 50% { background-color: yellow; } 100% { background-color: green; } }

Key Properties of CSS Animations

1. `@keyframes`

Defines the frames of the animation. You specify the starting and ending points (`from` and `to`) or intermediate steps (using percentages). 

@keyframes exampleAnimation {
    from { background-color: red; }
    to { background-color: yellow; }
}

2. `animation-name`

Specifies the name of the `@keyframes` animation that you want to apply to an element. 

animation-name: exampleAnimation;
Hover me!

3. `animation-duration`

Defines how long the animation takes to complete one cycle. 

animation-duration: 4s;
Duration Example

4. `animation-timing-function`

Specifies the speed curve of the animation. Values include `linear`, `ease`, `ease-in`, `ease-out`, `ease-in-out`, `cubic-bezier(x1, y1, x2, y2)`, and `steps()`. 

animation-timing-function: ease-in-out;
Timing Function Example

5. `animation-delay`

Specifies a delay before the animation starts. 

animation-delay: 2s;
Delay Example

6. `animation-iteration-count`

Specifies the number of times the animation should run. Values include a specific number (e.g., `2`), `infinite` (runs forever). 

animation-iteration-count: infinite;
Iteration Count Example

7. `animation-direction`

Specifies whether the animation should play in reverse on alternate cycles. Values include `normal`, `reverse`, `alternate`, and `alternate-reverse`. 

animation-direction: alternate;
Direction Example

8. `animation-fill-mode`

Specifies how a CSS animation should apply styles to its target before and after it is executing. Values include `none`, `forwards`, `backwards`, and `both`. 

animation-fill-mode: forwards;
Fill Mode Example

9. `animation-play-state`

Specifies whether the animation is running or paused. Values include `running` and `paused`. 

animation-play-state: paused;
Play State Example
