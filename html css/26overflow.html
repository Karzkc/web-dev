overflow property -

The overflow property specifies how to handle content that exceeds the boundaries of an element's box. It can have the following values:

  • visible
  • hidden
  • scroll
  • auto
  • inherit
In this example, we will use the "scroll" value to make the text inside the paragraph scrollable if it exceeds the boundaries of its box.

The default value is "visible".

overflow scroll means we can scroll

overflow auto - If the content is too long to fit within the box, it will cause a scrollbar to appear else it will not.

overflow inherit - It will inherit the overflow property from its parent element.

overflow none - This will hide the overflowing content, but it will still take up space in the layout.

overflow visible - This will show the overflowing content, but it will still take up space in the layout.

overflow hidden - This will hide the overflowing content, and it will not take up space in the layout.

white space - nowrap This will prevent text from wrapping onto new lines.

text-overflow - ellipsis - This will add ellipsis (...) at the end of the overflowed text to indicate that it has been truncated.
This property is used together with overflow property to show an ellipsis when text overflows the box.

clip -- This property is used to hide overflowing text. The text is not displayed, but it still takes up space in the layout.

overflow x and y - This property allows you to control the overflow behavior in both horizontal and vertical directions. syntax - overflow - x prop y prop eg - overflow: scroll x auto y hidden;
