CSS Filters

CSS filters are a powerful tool that allow you to manipulate the visual properties of an element. They are often used for creating cool effects, making images more dynamic, or adding visual effects to your website.

Some common CSS filters include blur, brightness, contrast, grayscale, hue-rotate, invert, opacity, saturate, sepia, and url().

To apply a CSS filter to an element, you can use the filter property, which accepts a comma-separated list of filter functions.

blur

The blur filter blurs the edges of an image. The amount of blur can be controlled by adjusting the value of the blur function.

Example: blur(5px)

brightness

The brightness filter adjusts the brightness of an image. The amount of brightness can be controlled by adjusting the value of the brightness function.

Example: brightness(2)

contrast

The contrast filter adjusts the contrast of an image. The amount of contrast can be controlled by adjusting the value of the contrast function.

Example: contrast(1.5)

grayscale

The grayscale filter converts an image to a grayscale version.

Example: grayscale(1)

hue-rotate

The hue-rotate filter rotates the hue of an image. The amount of rotation can be controlled by adjusting the value of the hue-rotate function.

Example: hue-rotate(45deg)

invert

The invert filter inverts the colors of an image.

Example: invert(1)

opacity

The opacity filter adjusts the opacity of an element. The amount of opacity can be controlled by adjusting the value of the opacity function.

Example: opacity(0.5)

saturate

The saturate filter adjusts the saturation of an image. The amount of saturation can be controlled by adjusting the value of the saturate function.

Example: saturate(2)

sepia

The sepia filter adds a sepia tint to an image.

Example: sepia(1)

drop shadow

The drop-shadow filter adds a drop shadow to an element. The shadow can be adjusted by adjusting the values of the drop-shadow function.

Example: drop-shadow(2px 4px 8px rgba(0, 0, 0, 0.2))
