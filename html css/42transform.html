Transform property

The transform property in CSS is used to manipulate an element's position, size, and shape. It is a powerful tool for creating complex designs.

rotate() deg or turn

The rotate() function rotates an element around its center by a given number of degrees. It accepts both positive and negative values. 

            transform: rotate(45deg);
            transform: rotate(0.25turn)
box

rotate x /y/z

The rotateX() and rotateY() functions rotate an element around the horizontal and vertical axes, respectively. They accept both positive and negative values. 

            transform: rotateX(45deg);
            transform: rotateY(45deg)
box

scale ,scale 3d xyz

The scale() function scales an element horizontally and vertically. It accepts both positive and negative values. 

            transform: scale(2);
            transform: scale(2, 1);
            transform: scaleX(2);
            transform: scaleY(2);
            transform: scaleZ(2);
            transform: scale3d(2, 2, 2);
box

skew xy notz

The skewX() and skewY() functions skew an element along the horizontal and vertical axes, respectively. They accept both positive and negative values. 

            transform: skewX(45deg);
            transform: skewY(45deg);
box

translate

The translate() function moves an element along the horizontal and vertical axes. It accepts both positive and negative values. 

            transform: translate(20px, 10px);
            transform: translateX(20px);
            transform: translateY(10px);
            transform: translateZ(20px);
box

transform origin

The transform-origin property sets the origin for transformations. It accepts values in the form of x-axis (left, center, right), y-axis (top, center, bottom), and z-axis (default is 0). 

            transform-origin: 50% 50%;
            transform-origin: left bottom;
            transform-origin: 20px 20px;
