Transition in css

In CSS, a transition allows you to change property values smoothly (over a given duration) rather than instantly. This is particularly useful for creating interactive elements like buttons or links where you want to add effects like changing color, size, or position when hovered or clicked. 


        
            selector {
                transition: property duration timing-function delay;
              }

Transition Properties -

1. `transition-property`

Definition: Specifies the name of the CSS property to which the transition is applied.

  • all: Applies the transition to all properties.
  • property-name: Specifies a single property (e.g., width, height, background-color).
  • none: No properties will transition.
        
transition-property: background-color;

2. `transition-duration`

Definition: Defines how long the transition takes to complete.

  • time: Duration in seconds (e.g., 0.5s) or milliseconds (e.g., 500ms).
  • Note: The default value is 0s (no transition).
        
transition-duration: 1s;
Hover me!

3. `transition-timing-function`

Definition: Describes how the intermediate values of the transition are calculated. Essentially, it controls the speed of the transition.

  • ease: Starts slowly, then speeds up, and ends slowly. (default)
  • linear: Maintains a constant speed from start to finish.
  • ease-in: Starts slowly and then accelerates.
  • ease-out: Starts quickly, then slows down.
  • ease-in-out: Starts slowly, accelerates in the middle, and slows down again.
  • steps(number, direction): Splits the transition into number steps. The direction can be start or end.
  • cubic-bezier(x1, y1, x2, y2): Defines a custom timing function with control points.

transition-timing-function: ease-in-out;

4. `transition-delay`

Definition: Specifies the time to wait before starting the transition.

  • time: Delay in seconds (e.g., 0.5s) or milliseconds (e.g., 500ms).
  • Note: The default value is 0s (no delay).
        
            transition-delay: 0.2s;

5. `transition` (Shorthand)

Definition: A shorthand for setting the four individual transition properties at once: transition-property, transition-duration, transition-timing-function, and transition-delay.

  • Syntax:
    •             
transition: property duration timing-function delay;
  • Example:
    •             
transition: background-color 1s ease-in-out 0.2s;
