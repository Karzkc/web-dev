grid line , grid cell , grid track ,grid area -

fundamental concepts in CSS Grid layout:

  • Grid line: The dividing lines that make up the grid structure.
  • Grid cell: The spaces between these lines, containing individual items.
  • Grid track: The horizontal and vertical lines that make up the grid structure, including the grid lines.
  • Grid area: The rectangular areas formed by four grid lines, encompassing multiple grid cells.

grid template areas and grid area

grid-template-areas: "header" "header" "sidebar" "main";
This means that - header area is named header and main area is named main.
grid-area: header;

fr unit

grid-template-columns: 1fr 1fr 1fr;
This means that - first column will take 1/3 of the grid space.
grid-template-rows: 1fr 1fr 1fr;
This means that - first row will take 1/3 of the grid space.
grid-area: main;
This means that - main area will take 2/3 of the grid space.
fr means - fractional units.
repeat(5,1fr)- means 5 times 1fr unit.
grid-template-columns: repeat(4, minmax(100px, 1fr)); This means that - the grid will have 4 columns with minimum width of 100px and maximum width of 1fr.

grid gap , row coulmn gap

grid-gap: 20px;
This means that - there will be a 20px gap between the grid cells.
grid-row-gap: 20px;
This means that - there will be a 20px gap between the grid rows.
grid-column-gap: 20px;
This means that - there will be a 20px gap between the grid columns.

justify/ align/ place - items /content /self

justify-items: center;
This means that - the grid items will be horizontally aligned to the center.
align-items: center;
This means that - the grid items will be vertically aligned to the center.
place-items: center;
This means that - the grid items will be horizontally and vertically aligned to the center.
justify-content: center;
This means that - the grid will be horizontally aligned to the center.
align-content: center;
This means that - the grid will be vertically aligned to the center.
place-content: center;
This means that - the grid will be horizontally and vertically aligned to the center.
