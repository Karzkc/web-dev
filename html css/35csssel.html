.box:first-child

The :first-child selector targets the first child element within its parent. The rule .box:first-child { background-color: red; } would apply the style to the first element with the class .box inside its parent, changing its background color to red.

.box::first-line

The ::first-line pseudo-element selector targets the first line of text within the selected element. The rule .box::first-line { color: yellow; } would change the color of the first line of text inside any .box element to yellow.

.boxes *

The universal selector * targets all elements. The rule .boxes * { color: blue; border: 2px solid black; } would apply a blue color and a black border to every element inside any element with the class .boxes.

p, a, .box, [data-color="primary"]

This selector group targets <p>, <a>, any element with the class .box, and any element with the attribute data-color="primary". The rule p, a, .box, [data-color="primary"] { padding-top: 45px; } would add a padding of 45px to the top of all these elements.

.box:nth-child(odd)

The :nth-child(odd) selector targets elements that are odd-numbered children within their parent. The rule .box:nth-child(odd) { background-color: blue; } would apply a blue background to every odd-numbered .box element within its parent.

.box:nth-last-child(2)

The :nth-last-child(2) selector targets the second-to-last child of a parent. The rule .box:nth-last-child(2) { background-color: red; } would apply a red background to the second-to-last .box element within its parent.

.boxes::before

The ::before pseudo-element selector inserts content before the content of the selected element. The rule .boxes::before { content: "Harry is good"; color: blue; } would insert the text "Harry is good" before the content of any element with the class .boxes, with blue text color.

.boxes::after

The ::after pseudo-element selector inserts content after the content of the selected element. The rule .boxes::after { content: "Sigma course is also good"; color: red; } would insert the text "Sigma course is also good" after the content of any element with the class .boxes, with red text color.

::selection

The ::selection pseudo-element selector targets the portion of an element that is selected by the user. The rule ::selection { background-color: black; color: aqua; } would change the background color to black and text color to aqua for any text that the user selects.

.box::first-letter

The ::first-letter pseudo-element selector targets the first letter of the selected element. The rule .box::first-letter { color: peru; font-size: 45px; } would change the color of the first letter of any .box element to peru and increase its font size to 45px.

input::placeholder

The ::placeholder pseudo-element selector targets the placeholder text of an input field. The rule input::placeholder { color: rgb(84, 84, 88); } would change the color of the placeholder text inside any input element to the specified RGB color.
