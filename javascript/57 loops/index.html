Javascript loops

loops - loops are used to execute a block of code repeatedly. They are essential for performing tasks repeatedly, such as iterating through arrays, counting, or performing calculations.
JavaScript has three types of loops: for, while, and do-while.


for loop

for loop is used to - iterate through a specified number of times.
in for loop there are three types: for , for in , for of

  • for loop - syntax - for (initialization; condition; increment/decrement) { statement(s); } 

    •             
                for (var i = 0; i < 5; i++) {
                    console.log('Number is'+ i);
                }
  • for in loop - syntax - for (variable in object) { statement(s); } 

    •             
                var arr = ['apple', 'banana', 'cherry'];
                for (var i in arr) {
                    console.log(arr[i]);
                }
  • for of loop - syntax - for (variable of iterable) { statement(s); } 

    •             
                var fruits = ['apple', 'banana', 'cherry'];
                for (var fruit of fruits) {
                    console.log(fruit);
                }

difference betweeen these 3 -
for loop - used when you know exactly how many times you want to iterate
for in loop - used when you want to iterate through the properties of an object
for of loop - used when you want to iterate through the values of an iterable object (such as an array or string)
Note: for of loop is more efficient than for in loop when you need to iterate through the values of an array or string.


while loop

while loop is used to - execute a block of code repeatedly as long as a specified condition is true.
syntax - while (condition) { statement(s); } 


        
            var i = 0;
            while (i < 5) {
                console.log('Number is'+ i);
                i++;
            }

difference between while loop and for loop -
while loop - used when you don't know exactly how many times you want to iterate
for loop - used when you know exactly how many times you want to iterate
Note: while loop is more flexible and easier to use when you don't know the exact number of iterations.


do-while loop

do-while loop is used to - execute a block of code once, and then repeatedly execute the block as long as a specified condition is true.
syntax - do { statement(s); } while (condition); 


        
            var i = 0;
            do {
                console.log('Number is'+ i);
                i++;
            } while (i < 5);

difference between do-while loop and while loop -
do-while loop - used when you want to execute the block at least once
while loop - used when you don't want to execute the block at least once
Note: do-while loop is more flexible and easier to use when you want to execute the block at least once.
Also nowadays do-while loop is not used as most developers prefer to use for, while, or for of loops.
