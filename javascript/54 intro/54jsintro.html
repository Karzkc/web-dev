about javascript

JavaScript is a programming language that adds interactivity to web pages. It can be used to create dynamic and interactive web pages.


we will discuss some js functions

alert function

This function displays a popup alert box with a specified message.

prompt function

This function displays a popup input box, allowing the user to input text.

confirm function

This function displays a popup confirmation box with two buttons: OK and Cancel.

document.write function

This function writes HTML code to a document.

document.title

This property sets or returns the title of the document.

document style

This property allows you to style HTML elements using CSS. syntax - document.style.property = value; example - document.style.color = "red";
