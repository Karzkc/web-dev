React Conditional Rendering & State Management

1. Introduction

This documentation explains React concepts like useState, conditional rendering, and .map() for rendering dynamic lists.

2. useState Hook

The useState hook is used to manage state in functional components.

const [count, setCount] = useState(0);

Here, count holds the state value, and setCount updates it.

3. Conditional Rendering

React provides multiple ways to conditionally render elements:

3.1 Using Logical AND (&&)

{toggle && }

The button appears only if toggle is true.

3.2 Using Ternary Operator

{toggle ?  : }

This renders a different button based on toggle's value.
