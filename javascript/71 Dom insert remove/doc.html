JavaScript DOM Manipulation Documentation

This documentation covers the various ways to manipulate the DOM using JavaScript, including querying elements, modifying attributes, and creating or removing elements. Below are the explanations and examples for each method.

Querying Elements

1. document.querySelector()

Finds the first matching element based on the selector provided. This method can be used with class names, tag names, or attribute selectors.

document.querySelector('.container');
Example: Selects the first element with the class .container.

2. innerHTML vs innerText

innerHTML retrieves or sets the HTML content inside an element, while innerText retrieves or sets only the text content without the HTML tags.

document.querySelector('.box').innerHTML;  // Returns HTML content
document.querySelector('.box').innerText;  // Returns only text

3. outerHTML vs outerText

outerHTML retrieves or sets the HTML of the element including the element itself, while outerText retrieves or sets the text of the element including the element itself.

document.querySelector('.container').outerHTML;
document.querySelector('.container').outerText;

Attributes and Properties

1. Accessing and Modifying Attributes

You can use the following methods to interact with element attributes:

  • hasAttribute(): Checks if the element has the specified attribute.
  • getAttribute(): Retrieves the value of an attribute.
  • setAttribute(): Sets the value of an attribute.
  • removeAttribute(): Removes the specified attribute from an element.
document.querySelector('h1').hasAttribute('style');  // Returns true/false
document.querySelector('h1').getAttribute('style');  // Returns attribute value
document.querySelector('h1').setAttribute('style', 'color: red');  // Sets new style
document.querySelector('h1').removeAttribute('style');  // Removes style attribute

2. Dataset Attributes

Custom data attributes (like data-createdby="kartik") can be accessed using the dataset property. This stores data attributes in a DOMStringMap object.

document.querySelector('.box').dataset;  // Returns DOMStringMap object
Example: Access the custom data attribute data-createdby: 
document.querySelector('.box').dataset.createdby;  // Returns 'kartik'

Inserting and Removing Elements

1. Creating New Elements

You can dynamically create new elements in the DOM using createElement and insert them using append, prepend, before, and after.

let newDiv = document.createElement('div');
newDiv.innerHTML = 'New content added';
document.querySelector('.container').append(newDiv);
Example: Creates a new div element with content and appends it to the container.

2. insertAdjacentHTML()

This method inserts HTML code at specified positions relative to the element. The positions can be:

  • beforebegin: Before the element.
  • afterbegin: Just inside the element, before its first child.
  • beforeend: Just inside the element, after its last child.
  • afterend: After the element.
let container = document.querySelector('.container');
container.insertAdjacentHTML('afterbegin', '
Added at the start
');
container.insertAdjacentHTML('beforeend', '
Added at the end
');
Example: Inserts HTML content inside the container at the beginning and the end.

3. Removing Elements

You can remove an element using the remove() method:

let elementToRemove = document.querySelector('.remove');
elementToRemove.remove();

Class Manipulation

1. classList Methods

The classList property allows you to add, remove, or toggle classes on elements. It returns a DOMTokenList of all classes assigned to the element.

document.querySelector('.container').classList;  // Returns a list of all classes
  • add(): Adds a new class to the element.
  • remove(): Removes a class from the element.
  • toggle(): Toggles a class on or off (if present, removes it; if absent, adds it).
document.querySelector('.container').classList.add('new-class');
document.querySelector('.container').classList.remove('old-class');
document.querySelector('.container').classList.toggle('active');

Document Editing Mode

1. document.designMode

When document.designMode is set to "on", you can edit the entire page as if it were a text document.

document.designMode = "on";
Note: This is often used for debugging or modifying web pages for quick edits.
