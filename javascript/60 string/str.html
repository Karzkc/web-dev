JavaScript String Methods

Strings are sequences of characters used for storing and manipulating text in JavaScript. Here are some commonly used string methods:

Accessing Characters

You can access individual characters in a string using square brackets: 

        
            let a = "hello";
            console.log(a[0]); // Outputs: h

Length

The length property returns the length of a string: 

        
            console.log(a.length); // Outputs: 5

Concatenation

Strings can be concatenated using the + operator or template literals: 

        
            let b = "world";
            console.log("First word is " + a + " and second is " + b); // Using +
            console.log(`First word is ${a} and second is ${b}`); // Using template literals

Escape Sequences

Escape sequences allow you to include special characters in strings. For example: 

        
            console.log("I\'m Kartik"); // Outputs: I'm Kartik

String Methods

  • toUpperCase(): Converts a string to uppercase.
  • toLowerCase(): Converts a string to lowercase.
  • slice(): Extracts a section of a string and returns it as a new string.
  • replace(): Replaces the first occurrence of a substring with another substring.
  • concat(): Concatenates two or more strings.
  • trim(): Removes whitespace from both ends of a string.
  • charAt(): Returns the character at the specified index.
  • indexOf(): Returns the index of the first occurrence of a specified substring.
  • startsWith(): Checks if a string starts with a specified substring.
  • endsWith(): Checks if a string ends with a specified substring.
        
            let c = " im kartik hi hello";
            console.log(c.toUpperCase()); // Outputs:  IM KARTIK HI HELLO
            console.log(c.toLowerCase()); // Outputs:  im kartik hi hello
            console.log(c.slice(1, 4)); // Outputs: im

            console.log(c.trim()); // Outputs: im kartik hi hello
