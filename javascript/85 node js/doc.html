Node.js and NPM Documentation

Introduction to Node.js

Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine. It allows developers to build server-side applications using JavaScript. With Node.js, you can handle backend logic, process requests, and manage files and databases.

Creating a Simple Server in Node.js

Below is an example of how to create a simple HTTP server in Node.js using the http module.

const { createServer } = require('node:http');

const hostname = '127.0.0.1';
const port = 3000;

const server = createServer((req, res) => {
  res.statusCode = 200;
  res.setHeader('Content-Type', 'text/plain');
  res.end('Hello World');
});

server.listen(port, hostname, () => {
  console.log(`Server running at http://${hostname}:${port}/`);
});

Introduction to NPM

NPM (Node Package Manager) is a tool that comes with Node.js to help you install, manage, and use external libraries or packages. You can use it to manage the dependencies in your projects.

How to Use NPM

To start using NPM, initialize your project using:

npm init

This command generates a package.json file where project details and dependencies are stored.

Example of a package.json File

Below is an example of a simple package.json file that stores your project's metadata, scripts, and dependencies:

{
  "name": "kartik",
  "version": "1.0.0",
  "description": "Amazing package, new to JS",
  "main": "index.js",
  "scripts": {
    "test": "echo 'Error: no test specified' && exit 1"
  },
  "author": "Kartik",
  "license": "ISC",
  "dependencies": {
    "express": "^4.21.1",
    "slugify": "^1.6.6"
  }
}

Installing Libraries via NPM

To install a library via NPM, use the command:

npm install [library-name]

For example, to install express or slugify:

npm install express
npm install slugify

Using Libraries in Your Code

Once you install libraries, you can import them into your code using require. Here's how you can use slugify:

let slugify = require('slugify');

let a = slugify("my name is kartik", ",");
console.log(a);

let b = slugify("@#$%^&*", ",");
console.log(b);

Popular NPM Packages

Some popular packages you may use frequently in Node.js projects are:

  • Express: A minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications.
  • Lodash: A modern JavaScript utility library that delivers modularity, performance, and extras.
  • Slugify: A package used to generate URL-friendly slugs from strings, useful for URLs and filenames.

Conclusion

Node.js combined with NPM is a powerful tool for building modern applications. By using packages and modules, you can extend the functionality of your projects with minimal effort, allowing you to focus on your application's logic and development.
