Node.js FS and Path Modules Documentation

The Node.js FS (File System) module provides functionality to interact with the file system, while the Path module handles and manipulates file paths. These modules are essential for working with files and directories in Node.js.

1. Introduction to the FS Module

The FS (File System) module allows you to perform various file operations, such as reading, writing, appending, and deleting files.

Here’s how you import the fs module:

const fs = require('fs');

2. Writing Files

To create or write to a file, you can use fs.writeFileSync() (synchronous) or fs.writeFile() (asynchronous):

fs.writeFileSync('File.txt', 'This is a new file with some text');

fs.writeFile('File2.txt', 'This is also a new file', (err) => {
  if (!err) console.log('Write successful!');
});

3. Reading Files

Use fs.readFile() (asynchronous) or fs.readFileSync() (synchronous) to read files:

fs.readFile('File2.txt', (err, data) => {
  if (!err) console.log(data.toString());
});

4. Appending Data to Files

The fs.appendFile() method allows you to add content to an existing file without overwriting it:

fs.appendFile('File2.txt', '\\nAppended line', (err) => {
  if (!err) console.log('Append successful!');
});

5. Deleting Files

To delete files, you can use fs.unlink():

fs.unlink('File2.txt', (err) => {
  if (!err) console.log('File deleted');
});

6. FS Module Additional Methods

Other important FS module methods include:

  • fs.mkdir(): Create a new directory.
  • fs.rmdir(): Remove a directory.
  • fs.rename(): Rename a file.
  • fs.stat(): Get file statistics.

7. Introduction to the Path Module

The Path module allows you to work with file paths and helps in manipulating and resolving paths efficiently. Import the path module as follows:

const path = require('path');

8. Working with File Paths

Some useful methods in the path module:

  • path.extname(): Returns the extension of the file.
  • path.basename(): Returns the last portion of a path (file name).
  • path.dirname(): Returns the directory name of a path.
  • path.join(): Joins multiple segments into a single path.
let filePath = 'D:/Projects/file.js';

console.log(path.extname(filePath)); // .js
console.log(path.basename(filePath)); // file.js
console.log(path.dirname(filePath)); // D:/Projects
console.log(path.join('D:/', 'Projects', 'file.js')); // Joins path segments

9. Practical Examples of FS and Path Modules

Here’s how you can combine the fs and path modules for various tasks:

const fs = require('fs');
const path = require('path');

let filePath = path.join(__dirname, 'File.txt');
fs.writeFile(filePath, 'Hello, world!', (err) => {
  if (!err) {
    fs.readFile(filePath, (err, data) => {
      if (!err) console.log(data.toString());
  });
}
});

10. Summary

The FS module provides powerful functionality to read, write, and manipulate files, while the Path module simplifies file path management. Both are foundational for handling files in Node.js.
