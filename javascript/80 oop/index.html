JavaScript Object-Oriented Programming (OOP)

In JavaScript, OOP allows us to create objects and define their structure using classes and constructors. Let's walk through the key OOP concepts:

1. Object Creation

An object in JavaScript is simply a collection of key-value pairs. Here's an example: 

let car = {
    "brand": "Tesla",
    "year": 2023
};
console.log(car);

This creates an object car with properties brand and year.

2. Constructor

A constructor is a special method that gets called when a new object is created from a class. It is used to initialize properties of the new object. Example: 

class Vehicle {
    constructor(brand) {
        this.brand = brand;
        console.log("Vehicle created!");
    }
}
let myCar = new Vehicle("Tesla");
console.log(myCar.brand);

The constructor in the Vehicle class sets the brand property when a new object is created using new.

3. Getters and Setters

Getters and setters are special methods to get and set the values of object properties. Example: 

class User {
    constructor(name) {
        this._name = name;
    }
    get name() {
        return this._name;
    }
    set name(value) {
        if (value.length < 3) {
            console.log("Name is too short!");
            return;
        }
        this._name = value;
    }
}
let user = new User("Kartik");
console.log(user.name); // "Kartik"
user.name = "Jo"; // "Name is too short!"

Getters and setters are defined using get and set keywords.

4. Inheritance

Inheritance allows one class to extend the properties and methods of another class. Example: 

class Animal {
    constructor(type) {
        this.type = type;
    }
    speak() {
        console.log(`${this.type} makes a sound.`);
    }
}
class Dog extends Animal {
    constructor(type, breed) {
        super(type); // Calls the parent constructor
        this.breed = breed;
    }
    bark() {
        console.log(`${this.breed} is barking.`);
    }
}
let myDog = new Dog("Mammal", "Labrador");
myDog.speak(); // "Mammal makes a sound."
myDog.bark();  // "Labrador is barking."

Here, Dog extends the Animal class and inherits its speak method.

5. Prototype and __proto__

JavaScript objects have a hidden __proto__ property, which links to another object (its prototype). Example: 

let electronics = {
    poweredOn: true
};
let phone = {
    brand: "Apple"
};
phone.__proto__ = electronics;
console.log(phone.poweredOn); // true

The phone object inherits the poweredOn property from the electronics object using the __proto__ link.

6. Instance of Operator

The instanceof operator checks if an object is an instance of a class or a constructor. Example: 

let isDog = myDog instanceof Dog; // true
let isAnimal = myDog instanceof Animal; // true

This shows that myDog is an instance of both Dog and Animal.

Conclusion

JavaScript OOP concepts such as constructors, inheritance, and prototype chains allow for creating reusable and scalable code. These concepts are crucial in organizing and structuring code in complex applications.
