MongoDB CRUD Operations

This documentation explains CRUD (Create, Read, Update, Delete) operations in MongoDB, step-by-step, with examples and best practices.

Create

Use the insertOne or insertMany methods to create new documents in a collection.

Example: Insert One

use("CRUD_DB");
db.createCollection("New_Courses");
db.New_Courses.insertOne({
    "name": "JavaScript",
    "language": "JS"
});
console.log("Data inserted!");

Example: Insert Many

db.New_Courses.insertMany([
    {"name": "Python", "language": "Python"},
    {"name": "HTML", "language": "HTML"}
]);
console.log("Multiple data inserted!");

Note: insertMany returns an array of inserted document IDs.

Read

Use find or findOne to retrieve data from a collection.

Example: Find All

let allDocs = db.New_Courses.find();
console.log(allDocs);

Example: Find One

let singleDoc = db.New_Courses.findOne({"name": "Python"});
console.log(singleDoc);

Tip: Use .count() to count documents in a result set.

Update

Use updateOne or updateMany to modify existing documents.

Example: Update Many

db.New_Courses.updateMany({
    "name": "Python"
}, {
    $set: { "name": "Updated Python" }
});
console.log("Documents updated successfully!");

Always verify your update query to avoid unintended changes.

Delete

Use deleteOne or deleteMany to remove documents.

Example: Delete Many

db.New_Courses.deleteMany({
    "name": "Updated Python"
});
console.log("Documents deleted successfully!");

Deleting documents is irreversible. Use caution while running delete commands.

Important Notes

  • MongoDB commands can be repeated if run multiple times, so double-check before executing them.
  • CRUD operations are the foundation of interacting with any database.
  • Always back up your database before running major operations like updates or deletions.

Summary

This documentation covers the basic CRUD operations in MongoDB. With proper understanding and precautions, MongoDB provides a powerful and flexible way to manage data efficiently.
