MongoDB Documentation

Introduction to MongoDB

MongoDB is a NoSQL database that is highly scalable and flexible. It stores data in collections of JSON-like documents, making it suitable for handling large amounts of unstructured data. If you're familiar with JavaScript, MongoDB’s JSON format and syntax make it particularly easy to learn.

Using MongoDB Compass

MongoDB Compass is a graphical interface for MongoDB that allows users to visualize, manage, and query their data without writing commands in a terminal.

Steps to Connect MongoDB Compass:

  1. Open MongoDB Compass and click "New Connection."
  2. Enter the connection string for your MongoDB server (e.g., mongodb://localhost:27017).
  3. Click "Connect" to view all available databases.

Common Compass Features

  • View Collections: See all documents in a collection, filter data, and apply projections.
  • Indexes: Create and manage indexes to optimize query performance.
  • Schema Analysis: Analyze document structure within a collection.

Using MongoDB in VSCode with MongoDB Extension

The MongoDB extension in Visual Studio Code enables you to work with MongoDB directly in the editor. You can connect to databases, query data, and use MongoDB Playground for interactive commands.

Connecting to MongoDB in VSCode:

  1. Install the MongoDB for VSCode extension.
  2. Open the MongoDB panel in VSCode and add a new connection.
  3. Enter the MongoDB connection string (e.g., mongodb://localhost:27017).
  4. Now you can browse databases and collections directly from VSCode.

MongoDB Playground

The MongoDB Playground is a feature within the VSCode MongoDB extension that lets you write and run MongoDB scripts interactively. This is especially useful for testing queries and managing data.

Example MongoDB Playground Script:

use('TEMP');
db.getCollection('courses').insertMany([ /* Insert courses here */ ]);

MongoDB Shell (mongosh)

mongosh is the MongoDB shell, which you can use to interact with the MongoDB server via command line. It’s a powerful tool for performing database operations directly.

Common mongosh Commands:

  • show dbs - Lists all databases
  • use <dbName> - Switches to a specific database
  • db.createCollection('collectionName') - Creates a new collection
  • db.collection.find() - Retrieves all documents in a collection
