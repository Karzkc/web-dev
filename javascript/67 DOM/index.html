DOM in JavaScript

The Document Object Model (DOM) represents the structure of a web page, allowing JavaScript to access and manipulate HTML elements dynamically.

1. Accessing DOM Elements

We can access various parts of the DOM using JavaScript methods like document.body and properties like childNodes.


// Access the body of the document
let bodyElement = document.body;

// Get child nodes (including text and comment nodes)
console.log(bodyElement.childNodes);

// Access specific child node by index
console.log(bodyElement.childNodes[1]);

// Get only element children (ignores text/comments)
console.log(bodyElement.children);

2. Manipulating DOM Elements

You can manipulate the DOM by changing element properties like style, innerText, and more.


// Change background color of the first box
let firstBox = document.body.firstElementChild.firstElementChild;
firstBox.style.backgroundColor = 'purple';

// Change text content of the last box
let lastBox = document.body.firstElementChild.lastElementChild;
lastBox.innerText = 'Modified Box';

3. Traversing the DOM

You can navigate through the DOM tree using properties like parentElement, nextElementSibling, and previousElementSibling.


// Get the parent of the first box
console.log(firstBox.parentElement);

// Access the next sibling element (Box 2)
console.log(firstBox.nextElementSibling);

// Access the previous sibling element of the second box
let secondBox = firstBox.nextElementSibling;
console.log(secondBox.previousElementSibling);

4. Working with Tables

In a table, you can access rows and cells using properties like rows and cells.

// Access the table and its rows
let table = document.querySelector('table');
console.log(table.rows);

// Access the first row
let firstRow = table.rows[0];
console.log(firstRow);

// Access individual cell
let firstCell = firstRow.cells[0];
console.log(firstCell);

5. Dynamic Creation of Elements

DOM allows you to create new HTML elements and insert them into the document dynamically.


// Create a new div element
let newDiv = document.createElement('div');
newDiv.className = 'box';
newDiv.innerText = 'New Box';

// Append the new div to the container
let container = document.querySelector('.container');
container.appendChild(newDiv);
