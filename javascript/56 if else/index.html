About if else statement and some assignment operators


1 - conditonal statement 


            
                if(condition){
                    // code to execute if condition is true
                    // ex: console.log('Condition is true');
                } else {
                    
                    // code to execute if condition is false
                    // ex: console.log('Condition is false');
                }

if else ladder - 
            
                if(condition1){
                    // code to execute if condition1 is true
                } else if(condition2){
                    // code to execute if condition1 is false and condition2 is true
                } else {
                    // code to execute if condition1 and condition2 are false
                }


2 - now some operators


    Mathematical oeprators -
  • + (addition)
  • - (subtraction)
  • * (multiplication)
  • / (division)
  • % (modulus) - gives remainder
  • ++ (increment) - increases the value by 1
  • -- (decrement) - decreases the value by 1
  • ** (exponentiation) - raises the first operand to the power of the second operand 
    •             
                var x = 10;
                var y = 5;
                console.log('x + y ='+ (x + y)); // x + y = 15
                console.log('x - y ='+ (x - y)); // x - y = 5
                console.log('x * y ='+ (x * y)); // x * y = 50
                console.log('x / y ='+ (x / y)); // x / y = 2
                console.log('x % y ='+ (x % y)); // x % y = 0
                console.log('x++ ='+ (x++)); // x++ = 11
                console.log('x ='+ x); // x = 11
                console.log('x-- ='+ (x--)); // x-- = 11
                console.log('x ='+ x); // x = 10
                console

    now assignment operators -
  • = (assignment)
  • += (addition and assignment)
  • -= (subtraction and assignment)
  • *= (multiplication and assignment)
  • /= (division and assignment)
  • %= (modulus and assignment)
  • **= (exponentiation and assignment)

    • now comparison operators -
  • == (equality)
  • === (strict equality) - checks for value and type
  • != (not equal)
  • !== (strict not equal) - checks for value and type
  • > (greater than)
  • < (less than)
  • >= (greater than or equal to)
  • <= (less than or equal to)
  • ternary operator - ? this will put the execution only when a certain condition is true

    • example of given oeprators - 
                
                var x = 10;
                var y = 5;
                console.log('x == y ='+ (x == y)); // x == y = false
                console.log('x === y ='+ (x === y)); // x === y = false
                console.log('x!= y ='+ (x!= y)); // x!= y = true
                console.log('x!== y ='+ (x!== y)); // x!== y = true
                console.log('x > y ='+ (x > y)); // x > y = false
                console.log('x < y ='+ (x < y)); // x < y = true
                console.log('x >= y ='+ (x >= y)); // x >= y = false
                console.log('x <= y ='+ (x <= y)); // x <= y = true
