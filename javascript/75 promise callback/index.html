A synchronous nature of js

JavaScript is single-threaded, meaning it executes code in a single sequential flow. This means that only one piece of code can be executed at a time, preventing concurrent execution.
This makes the execution of JavaScript code blocking, meaning that the execution of one script will halt until the other script has finished.
This can lead to performance issues, as it makes the browser unresponsive during script execution.
For example, consider the following code:
for (let i = 0; i < 1000000; i++) { console.log(i); }
In this code, the loop will take a long time to complete, making the browser unresponsive until it finishes.
To improve performance, asynchronous programming techniques can be used, such as callbacks, promises, or async/await
