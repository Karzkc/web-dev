vairables in js

definition - A variable is a named location in memory used to store data. It has a specific type and can be changed throughout the program. syntax - let variableName: data type = value;

var

var is function-scoped. It can be reassigned and redeclared within the same function. syntax - var variableName = value;

let

let is block-scoped. It cannot be reassigned and redeclared within the same block (i.e., a pair of curly braces {}). syntax - let variableName = value;

const

const is block-scoped. It cannot be reassigned and redeclared within the same block. It can be used to declare a constant value that cannot be changed. syntax - const variableName = value;

typeof

typeof operator returns the type of a variable. syntax - typeof variableName;

rules and what kind of variable names are allowed or not

- Variable names must start with a letter, an underscore (_), or a dollar sign ($).
- Variable names can contain letters, numbers, underscores (_), and dollar signs ($).
- Variable names are case-sensitive (e.g., myVariable and myvariable are different variables).
- Variable names cannot be reserved keywords (e.g., let, var, const, class, etc.).
- Variable names cannot start with a number.
- Variable names cannot contain any spaces.
- Variable names cannot contain any special characters other than underscore (_).
- Variable names should be meaningful and easy to understand.
- Variable names should not contain any consecutive underscores (_).
example - let myVariable = "Hello World!"; const PI = 3.14; let _underscore_variable = "Hello World!"; let $dollar_variable = "Hello World!"; let my_variable = "Hello World!"; // not allowed because it contains spaces let my-variable = "Hello World!"; // not allowed because it contains a hyphen let let = "Hello World!"; // not allowed because it is a reserved keyword

difference bw let var and const

- let is block-scoped and can be reassigned and redeclared within the same block.
- var is function-scoped and can be reassigned and redeclared within the same function.
- const is block-scoped and cannot be reassigned. It can be used to declare a constant value that cannot be changed.
example - let x = 10; { let x = 20; console.log(x); // Output: 20 } console.log(x); // Output: 10

Data Types & Objects

- JavaScript supports different data types such as numbers, strings, booleans, null, undefined, symbol, bigint, and object.
- Objects are used to store data and behavior in JavaScript. Objects are created using object literals or by using the `new` keyword followed by the constructor function.
example -

        let myBigInt = 10000000000000000000n;
        let myObject = {
        name: "John Doe",
        age: 30
        };
