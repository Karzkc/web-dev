JavaScript DOM and BOM Documentation

What is DOM?

The DOM (Document Object Model) is a programming interface for HTML and XML documents. It represents the document as a tree of objects that can be manipulated with JavaScript. Each HTML tag is an object in the DOM tree, and these objects can be accessed and modified using various DOM properties and methods.

DOM Example: Manipulating Document Properties

Let's start by changing the document title and styling the body:


// Change document title
document.title = "New Title";
console.log(document.title); // Logs the updated title

// Access body and head elements
console.log(document.body);  // Displays the content of the body
console.log(document.head);  // Displays the content of the head

// Change background color of the body
document.body.style.backgroundColor = "aqua";
Output after changing the title: "New Title"
Background color changes to aqua.

Accessing Selected HTML Element

The $0 variable in the browser console allows you to reference the currently selected element in the Elements panel of DevTools.


// Select the currently highlighted element in the browser console
$0.style.backgroundColor = "lightgreen";
The selected element's background color is changed to lightgreen.

What is BOM?

The BOM (Browser Object Model) allows JavaScript to interact with the browser window. It represents the browser environment, including the window, history, navigator, and other browser-related objects. Unlike the DOM, which focuses on the content of the webpage, the BOM is concerned with the browser and its features.

BOM Example: Accessing the Window Object

With the BOM, you can manipulate the browser window, such as resizing, opening a new window, or accessing various window properties.


// Access the window object
console.log(window.document);  // Displays the document object

// Resize the window
window.resizeTo(800, 600);

// Scroll the window
window.scrollBy(0, 300);  // Scroll down by 300 pixels

// Accessing the window's history
console.log(window.history.length);  // Logs the number of entries in the browser history
Output: Window resized to 800x600 and scrolled down by 300 pixels. Logs the browser history length.

Accessing Browser Information

You can use the navigator object to get information about the browser, such as the user's platform or the browser's version.


// Access browser details using navigator
console.log(window.navigator.userAgent);  // Logs the browser's user agent string
console.log(window.navigator.platform);   // Logs the platform (e.g., "Win32" for Windows)
Output: Logs the browser's user agent and platform (e.g., "Win32").
