CSS Sizing Units

  1. vw - viewport width percentage Example: height: 50vmax; Example: width: 50vw;
  2. vh - viewport height percentage Example: height: 50vmax; Example: height: 50vh;
  3. vmin - viewport width or height, whichever is smaller Example: height: 50vmax; Example: width: 50vmin;
  4. vmax - viewport width or height, whichever is larger Example: height: 50vmax; Example: width: 50vmax;
  5. % - percentage of the parent element or elemetn in which it exit eg div p span {...} Example: height: 50vmax; Example: width: 50%;
  6. em - font size of the parent element means how much times of inherit Example: height: 50vmax; Example: font-size: 1.5em;
  7. rem - font size of the root element (html) Example: height: 50vmax; Example: font-size: 1.5rem;
  8. px - pixel Example: height: 50vmax; Example: font-size: 18px;
